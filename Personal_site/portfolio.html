Throughout the course, have been assigned assignments and projects to be completed. Below explaining the process and responsbilites for each one.

First Project

  • Working in a group setting, the team suggests what's best for the code and discuss opinions on how to create the code.
  • Being the navigator for the group, while seeing what values will make the project look the best.
  • Present the site along with the team and explain every detail put into the code.

Second Project

  • Creating the project in a pair, deciding on the theme of the project and what's needed inside
  • Each person handling a certain part of the code for each part of the site, then discuss what will be needed or not.
  • Incorpatine the presentation and present to the instructor our theme and what went behind the whole process.

Third Project

  • Doing a solo project and use the previous experience from creating a code with others.
  • Create a plan for the project and work on each part seperately to make sure there are no errors.
  • Present the project to the instructor and class, talk about the details and the possible problems that may have occurred.
